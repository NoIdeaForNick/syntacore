Директории:
prj - проект для САПР quartus. Отчеты фиттера и анализатора таймингов
rtl - код синтезируемых модулей
scripts - скрипты для автоматизированного запуска тестов
tb - код тестбенчей (direct + crv). Основной тест - crv. Direct - вспомогательные прямые тесты для проверки отдельных модулей.

Примечания: 
1) В модуле cross bar выполнены два первых дополнительных задания: 4 мастера + 4 слейва, дополнительный сигнал resp. Оставшиеся пункты доп. заданий выполнить несложно, но нужны комментарии к логике работы. 
2) Для запуска автоматизированного теста, на компьютере должны быть установлены: tcl интерпретатор и QuestaSim. Запуск в Modelsim не является валидным, т.к. в тесте используются методы randomize, SVA и coverage. Проверка покрытия отключена по умолчанию, т.к. занимает некоторое продолжительное время (15-20 мин). ООП тест написан полностью на SV и не требует никаких дополнительных компонентов.
3) Проверку рабочей частоты осуществляю с помощью защелкивания всех входных/выходных сигналов (кроме тактового и ресета) в регистры. Сигналы с данных регистров являются virtual pin.
